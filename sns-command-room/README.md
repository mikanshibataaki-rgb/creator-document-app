# SNS司令室 Ver.1

岐阜スタジオ専用の、SNS投稿作業を短くするAIアシスタントです。

これは完全自動投稿アプリではありません。
投稿文を作る、コピーする、SNSを開く、投稿後URLを保存するところまでを助けます。

## まずできること

- `今日の投稿を作る` ボタンで投稿文を作る
- 投稿目的、対象、部屋、トーンを選ぶだけで使える
- X / Instagram / Facebook / Threads 用に文章を分けて作る
- Instagramにはハッシュタグを付ける
- 各SNS投稿文をコピーできる
- 各SNSを開くボタンがある
- 投稿後URLを保存できる
- 投稿履歴を見られる
- 履歴から `再投稿用に改善` できる
- OpenAI APIが使えない時も仮テンプレートで止まらない

## 今は必須にしないこと

- Facebook API連携
- Instagram API連携
- Meta API連携
- 完全自動投稿
- 反応数の細かい自動取得

## 初回準備

Node.jsを入れてから、このフォルダで必要な部品を入れます。

ターミナルが苦手な場合は、Finderで次をダブルクリックしてください。

```text
install.command
```

## 起動方法

Finderで次をダブルクリックしてください。

```text
start.command
```

ブラウザで次が開きます。

```text
http://localhost:3000
```

黒い画面が開いた場合、その黒い画面は閉じないでください。
黒い画面が開いている間だけアプリが動いています。

## 基本の使い方

1. アプリを開きます。
2. `投稿目的`、`対象`、`部屋`、`トーン` を選びます。
3. `今日の投稿を作る` を押します。
4. X / Instagram / Facebook / Threads の投稿文が出ます。
5. 使いたい文章の `コピー` を押します。
6. `Xを開く` などでSNSを開きます。
7. コピーした文を貼り付けて投稿します。
8. 投稿後、投稿URLを貼って `投稿URLを保存` を押します。

## OpenAI APIについて

OpenAI APIキーが設定されていると、AIが投稿文を作ります。

`.env.local` に次のように入れます。

```text
OPENAI_API_KEY=sk-...
```

APIキーがない、または利用上限に達した場合でもアプリは止まりません。
その場合は仮テンプレートの投稿文を表示します。

## Meta APIについて

Meta API連携は任意です。
設定しなくてもSNS司令室は使えます。
FacebookやInstagramのAPI設定は難しいため、Ver.1では必須機能から外しています。

## 保存される場所

投稿履歴はローカルの次のファイルに保存されます。

```text
data/posts.json
```

まずはローカルで使いやすくすることを優先しています。

# クリエイター案件マネージャー β

クリエイター案件の「職業選択 → 聞き取り → 案件整理 → 見積 → 精算 → 業務内容確認書 → 納品書 → 請求書」を共通データから管理し、A4 PDFへ出力するNext.jsアプリです。

## 起動

```bash
npm install
npm run dev
```

`http://localhost:3000` を開きます。入力内容と取引先はブラウザの `localStorage` に自動保存されます。

## 構成

- `domain/`: 型、税計算、初期値。UIや保存先に依存しません。
- `data/documentRepository.ts`: 保存境界。将来ここへSupabase実装を追加できます。
- `features/editor/`: 入力UI。
- `features/documents/`: 聞き取りシート、案件シート、見積書、精算書、業務内容確認書、納品書、請求書のA4帳票UI。
- `features/workspace/`: 画面状態、保存、タブ連携。

## 主な機能

- 最初の画面は複数選択できる「職業選択」。
- 選んだ職業に合わせて、聞き取り項目と見積テンプレートを切り替え。
- 見積明細の税区分は「税別 / 税込 / 非課税 / 不課税」。税込は内税、税別は外税として計算。
- 見積とは別に `expenseLines` で「精算・領収書」明細を管理。
- 請求書には制作費と精算費を分けて表示。
- 納品書PDFを追加。

## PDF

「PDF出力」からブラウザの印刷ダイアログを開きます。A4、倍率100%、余白なし、背景グラフィックありを推奨します。

# Creator Agreement

②クリエイター契約アプリです。

クリエイターと発注者が、口約束ではなく契約書・発注確認書のたたき台を作成できるアプリです。報酬、納期、納品物、修正回数、使用範囲、キャンセル条件など、仕事を始める前に決めておく内容を整理します。

## 注意

β版です。本サービスは契約書作成を支援するツールであり、法的効力や個別案件への適合性を保証するものではありません。重要な契約は専門家へご確認ください。入力内容はこの端末のブラウザに保存されます。

## 保存

現状はブラウザローカル保存です。サーバー保存、実メール送信、SMS認証、決済、デプロイは行っていません。

## 作業対象

このアプリの作業対象は `creator-agreement` です。

## 起動

```bash
npm install
npm run dev -- --hostname 127.0.0.1 --port 3101
```

ブラウザで `http://127.0.0.1:3101` を開きます。

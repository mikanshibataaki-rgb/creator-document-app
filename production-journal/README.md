# 制作日誌

Creator OS内で使う、制作現場の記録を「放り込む」ためのローカルMVPです。

## 起動

```bash
npm install
npm run dev
```

`http://localhost:3000` を開きます。

## 現在の範囲

- 仮ログイン
- 投稿、編集、削除、下書き
- 写真・映像の一時プレビュー
- 音声入力（対応ブラウザのみ）
- 公開・掲載・SNS共有・名前表示の許可設定
- ブラウザ内保存
- フィードと絞り込み
- ポイントとレベル
- 付箋カード
- 簡易絵巻
- 付箋単体表示

本物のログイン、DB、クラウドメディア保存、外部公開、外部API、AI機能は未実装です。
